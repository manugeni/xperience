## Export Basics

### Before Exporting
1. **Project Cleanup**
   - Remove unused assets
   - Hide unnecessary layers
   - Check scene optimization

2. **Save Your Project**
   - Create backup
   - Version your files
   - Check dependencies

### Export Process
1. Open Export dialog (File > Export)
2. Choose output format
3. Select export location
4. Configure settings
5. Start export

## Output Formats

### Video Formats
1. **MP4 (H.264)**
   - Best for web sharing
   - Good compression
   - Wide compatibility
   
   Settings:
   - Bitrate: 8-50 Mbps
   - Frame rate: 24-60 fps
   - Resolution: Up to 4K

2. **ProRes**
   - Professional quality
   - Large file size
   - Good for post-processing
   
   Settings:
   - Codec: ProRes 422/444
   - Color depth: 10-bit
   - Resolution: Up to 8K

3. **Raw Sequence**
   - Maximum quality
   - Frame-by-frame export
   - Perfect for compositing
   
   Formats:
   - PNG sequence
   - EXR sequence
   - TIFF sequence

### Audio Export
1. **Formats**
   - WAV (uncompressed)
   - AAC (compressed)
   - MP3 (compressed)

2. **Settings**
   - Sample rate: 44.1/48 kHz
   - Bit depth: 16/24-bit
   - Channels: Stereo/5.1

2. **Professional**

### Render Quality
1. **Draft**

   - Preview quality

2. **Standard**

3. **High Quality**
   - Best quality

## Platform-Specific Guidelines

### YouTube
- Resolution: 1080p/4K
- Format: MP4 (H.264)
- Bitrate: 15-50 Mbps
- Audio: AAC 48kHz

### Instagram
- Resolution: 1080x1080 (square)
- Format: MP4 (H.264)
- Length: Under 60 seconds
- Bitrate: 8-12 Mbps

### Professional Delivery
- Format: ProRes
- Resolution: Project specific
- Audio: WAV 48kHz
- Include handles
